\doxysubsection{Detailed Description}
Abstract class of the termination method. 

Since the termination method is a field of an optimization method and it needs information about it\textquotesingle{}s parent class, it\textquotesingle{}s main function takes as it\textquotesingle{}s argument a pointer to the optimization method whose field the termination method is.
